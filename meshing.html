Meshing¶

Mesher¶

Mesh of a sphere including cuboid shell for the solution of open-boundary problems. (a) Explosion view (b) Cross section

class magnumfe.Mesher¶

This class uses the Gmsh library to generate dolfin meshes. It provides methods to surround arbitrary meshes with a cuboid shell as required by the shell-transformation method, see Shell-Transformation Method. Moreover regular cuboid meshes can be created and domain information can be attached to the resulting dolfin mesh.

The shell is always created symmetrically around the coordinate origin. Hence mesh files read by this class should also be constructed symmetrically around the origin.

Example

# Read sample.msh and create shell
mesher = Mesher()
mesher.read_file("sample.msh")
mesher.create_shell(2, 0.1, (10, 10, 10));
mesh = mesher.mesh()

# Create cuboid mesh and surround with shell
mesher = Mesher()
mesher.create_cuboid((100.0, 100.0, 5.0), (20, 20, 1))
mesher.create_shell(2);
mesh = mesher.mesh()

create_shell(d, margin=0.0, n=(0, 0, 0), progression=1.0)¶

Creates a cuboid shell around the current sample.

Arguments
d (int)
Number of layers of the shell mesh.
margin (float)
Margin between sample and shell (useful for meshes read from file).
n ([int])
Number of mesh points of shell. (only required if mesh was read from file)
progression (float)
Defines coarsening of the shell layers. 1.0 means no coarsening.
